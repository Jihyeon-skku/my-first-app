## 📖 Description

**Hello SKKU** is a Streamlit-powered web dashboard designed to provide an interactive and visual experience for SKKU-related data and services. Built with simplicity and usability in mind, it leverages the power of Streamlit to deliver a responsive, browser-based interface with minimal setup.

---

## ✨ Features

- 📊 **Interactive Dashboard** — Real-time, dynamic data visualizations built with Streamlit
- 🖥️ **Browser-Based UI** — No installation headaches; runs entirely in your browser
- ⚡ **Fast & Lightweight** — Quick load times with a clean, minimal interface
- 🔧 **Easy Configuration** — Simple setup with a single Python file entry point
- 📱 **Responsive Design** — Works across desktop and mobile screen sizes
- 🔄 **Live Reload** — Instant updates as you modify the source code

## 🚀 How to Run

### Prerequisites

- Python 3.9 or higher
- pip package manager

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/hello-skku.git
cd hello-skku

# 2. (Optional) Create and activate a virtual environment
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows

# 3. Install dependencies
pip install -r requirements.txt
```

### Run the App

```bash
streamlit run app.py
```

Then open your browser and navigate to:

```
http://localhost:8501
```
